## Project Context:
Coden Portal is a web-based application designed to manage and display structured content and user-facing features within a centralized system. It includes multiple functional modules such as user interaction pages, content sections, and access-based components.

The application was tested to validate core workflows, UI behavior, and system functionality across different browsers and devices. Testing focused on ensuring stable navigation, correct data rendering, and proper handling of user interactions across modules.

This document contains manual test cases created to validate the functionality, usability, and reliability of key features within the platform.

The spreadsheet includes:
- QA Tracker
- Tracking and Summary
- Test scenarios
- Steps to reproduce
- Expected results
- Execution status
