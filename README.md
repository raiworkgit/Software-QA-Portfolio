# Software-QA-Portfolio
A collection of QA documentation from testing different projects, including bug reports, test cases, and supporting evidence such as screenshots and screen recordings. 
---------------------------------------------------------------------------------------

This repository contains QA artifacts created during manual testing of various web applications across different projects and platforms. It includes structured bug reports, sample test cases, and supporting evidence such as screenshots and screen recordings to demonstrate testing activities and defect validation.

## P.S.
Some files were reconstructed from available exported documents, screenshots, and personal records due to limited access to original project management tools.
